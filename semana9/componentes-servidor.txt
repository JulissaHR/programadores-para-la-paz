Sistema operativo:
Es el software principal del computador que se encarga de administrar los recursos del sistema y permitir la ejecución de aplicaciones como Node.js.

Node.js:
Es un entorno de ejecución que permite ejecutar código JavaScript fuera del navegador. Se utiliza para crear servidores y aplicaciones web.

Aplicación server.js:
Es el archivo que contiene la lógica del servidor. Define las rutas y las respuestas que recibirán los usuarios cuando accedan al servicio.

Puerto:
Es un punto de comunicación utilizado por el servidor para recibir solicitudes. En esta actividad se utiliza el puerto 3000.

Logs:
Son registros que almacenan eventos y actividades del servidor. Ayudan a monitorear el funcionamiento y detectar posibles errores.
